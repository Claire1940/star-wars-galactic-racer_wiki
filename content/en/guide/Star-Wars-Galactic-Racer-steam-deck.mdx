<Section icon={Gamepad} title="Controls and Input Configuration">

Star Wars Galactic Racer features a complex flight model involving Afterburners, Ramjets, and unique chassis traits like "Knife Edge" and "Kinetic Burst." Configuring your Steam Deck inputs correctly is vital for surviving the hazards of Jakku and Ando Prime.

<Steps>
  <Step num="1" title="Enable Gyro Steering">
    Navigate to the Steam Input settings and set Gyro to "As Joystick." This allows for finer banking maneuvers on Skim Speeders.
  </Step>
  <Step num="2" title="Map Back Paddles">
    Assign L4/R4 to Afterburner and L5/R5 to Ramjet. This keeps your thumbs on the sticks for steering and camera control.
  </Step>
  <Step num="3" title="Adjust Trigger Deadzones">
    Set the accelerator (R2) deadzone to 5% to ensure immediate launch boost timing during the race countdown.
  </Step>
</Steps>

| Action | Default Input | Recommended Deck Mapping |
| :--- | :--- | :--- |
| **Accelerate** | R2 | R2 / Right Trigger |
| **Brake / Drift** | L2 | L2 / Left Trigger |
| **Afterburner** | A / Cross | R4 (Back Paddle) |
| **Ramjet (Boost)** | B / Circle | R5 (Back Paddle) |
| **Shunt Left/Right** | LB / RB | L1 / R1 |
| **Takedown** | Y / Triangle | Y / Top Button |
